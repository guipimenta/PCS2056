INSTRUÇÕES PARA RODAS O PROJETO

LOCALIZAÇÃO ORIGINAL DO ARQUIVO ENTRADA.txt: analisador_lexico/ENTRADA.txt

OPÇÃO 1 (se possuir CodeBlocks instalado):
  Abra o projeto do CodeBlocks analisador_lexico.cbp (localizado na pasta analisador_lexico) e rode o programa pelo CodeBlocks.
OPÇÃO 2 (ambiente Linux):
  1. Abra o prompt de comando na raiz (AnalisadorLexico) e rode o comando "make"
  2. O programa deve rodar diretamente no prompt.
OPÇÃO 3:
  1. Crie um a pasta nova na raiz do projeto e crie um projeto novo em sua IDE
  2. Inclua o arquivo ENTRADA.txt na pasta do projeto da IDE
  3. Rode o programa pela sua IDE.
